# SimpleNoclip

## Support
Support this mod here, where it was originally uploaded: https://www.gta5-mods.com/scripts/simplenoclip

## Features:

- Simple intuitive controls, all movement can be done with left hand only.
- Controls are easily configurable, so if you don't like my setup that's okay!
- Variable noclip speed, by default can go up to 1000.0 (pretty fast!)
- UFO Mode feature, only allows vertical movement via up and down movement keys (Space & Ctrl by default)
- Displays what noclip mode you are in, and what speed you are currently doing.
- Hide the Radar, the rest of the HUD, and the script's UI render at anytime for screenshots or whatever purpose you desire!
- Precision Mode, allows you to go 10% your current speed easily with one key. (X by default)


## Controls:
- Movement: WASD
- Enter/Exit Noclip: N
- Enter/Exit UFO Mode: G
- Speed Boost: Shift
- Variable speed increase/decrease: Mouse scroll
- Move up vertically: Space
- Move down vertically: Ctrl
- Precision Mode: X
- Hide HUD: H

## Prerequisites:
- Latest version of ScriptHookV
- Latest version of Microsoft's cpp Redistributable

## How to install:
- Make sure you have prerequisites installed first!
- Simply move all files into GTA V root directory (Same directory ScriptHookV exists).
